## [0.1.0] - 2025-10-19

### Added

- 初始版本發布
- 新增 `plugin-dev:quick-fix` Skill
  - 在測試 plugin 過程中快速修正功能
  - 快速識別當前測試的 plugin 和相關檔案
  - 根據使用者回饋定位需要修改的 md 檔案
  - 立即進行修正並驗證
  - 提供修改前後的對比說明
  - 支援修改 Agent、Command 或 Skill 的定義檔
- 提供完整的使用場景和範例說明
- 協助快速迭代改善 plugin 品質
